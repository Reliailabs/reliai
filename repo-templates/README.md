# Repo Templates

This directory contains the initial public scaffolds for the planned `github.com/reliai` ecosystem.

Included templates:

- `reliai`
- `reliai-python`
- `reliai-demo`
- `reliai-examples`
- `reliai-rag-starter`
- `reliai-agent-starter`

These are starter repository contents only. They do not participate in the current platform runtime.
