# 1-Minute Reliai Elevator Demo

Use this when someone asks:

"So what does Reliai actually do?"

This version works for:
- hallway conversations
- conference booths
- investor intros
- short technical chats

## Script

### 0–10 seconds — Define the problem

"When teams ship AI systems to production, failures are hard to detect. Models start hallucinating, prompts regress, or latency spikes, and engineers usually find out from users."

### 10–25 seconds — Explain what Reliai is

"Reliai is the reliability control plane for AI systems. It automatically detects regressions in production, opens incidents, and traces exactly where the pipeline failed."

### 25–40 seconds — Explain how it works

"The SDK instruments every AI request across retrieval, prompts, model calls, and guardrails. Reliai analyzes those traces to detect reliability issues like hallucination spikes or prompt regressions."

### 40–55 seconds — Explain the value

"When something breaks, Reliai explains the root cause and recommends guardrails or fixes before the issue spreads to users."

### 55–60 seconds — Close with the outcome

"So instead of discovering AI failures from user complaints, teams catch them immediately and keep their AI systems reliable in production."

## One-Sentence Version

"Reliai detects AI regressions in production, explains why they happened, and recommends guardrails before users see the failure."

## Why This Pitch Works

It follows a structure engineers and investors can absorb quickly:

problem  
-> system  
-> mechanism  
-> outcome
