# Reliai 15-Second Pitch Variants

Use these when you need a short intro before moving into the control panel or 3-minute demo.

## 1. Default Founder Version

"Reliai is the reliability control plane for AI systems. It detects AI regressions in production, explains the root cause, and recommends guardrails before users see the failure."

Why it works:
- defines the category
- explains the function
- states the value

## 2. Investor Version

"Teams are shipping AI into production, but failures like hallucinations or prompt regressions are hard to detect. Reliai automatically detects those regressions, traces the root cause, and recommends fixes before users notice."

Why it works:
- starts with the market problem
- explains how Reliai solves it
- shows operational value

## 3. Engineer Version

"Reliai instruments AI pipelines with distributed tracing, detects reliability regressions in production, and recommends guardrails to keep AI systems stable."

Why it works:
- uses engineering language
- highlights the mechanism
- signals infrastructure positioning

## Simplest Possible Version

"Reliai detects AI failures in production and tells you how to fix them before users notice."

## Follow-Up Line

After any of these, the best next sentence is:

"The easiest way to understand it is to look at the control panel."
